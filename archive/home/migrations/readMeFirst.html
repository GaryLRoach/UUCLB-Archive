WELCOME TO THE UUCLB ARCHIVES

If you are a first time user of this site, please read this section in its entirety.

Public users:

We are sorry but at this time there are no records available for public viewing. The process of putting our records on line started in December 2015 and will not be completed for some time. Each record will then have to be reviewed – and maybe redacted – before being posted for public consumption. Check back periodically after about 6 months.

Member users:

You will have to have a personal password assigned to use this site. The passwords for the main web site will not work. Please contact the church historian – Gary Roach – personally or by email at historian.uuclb.org. (Subject to change)

The use of the site should be self explanatory but for the computer challenged the following might help:

If you got to this point, you must have found the “archive” button on the church's main web site.

Place the mouse pointer over the “Membership” button and left click. All if the buttons should go gray (deactivated) and the log-in page should appear. Input your email address of record and the password you obtained from the historian. If you receive an error, carefully recheck your entries. If you can't find the problem contact the historian.

Once logged in, you will be presented with a records search page. This is still in the works but will allow for recursive key word searches. Just keep feeding in new key words until the number of documents is reduced to a manageable size. Then select each document you wish to read. You will be able to print the files to your printer. We admonish you to keep this information inside the church community. We will not look kindly on anyone using the information inappropriately. When in doubt, ask.

Some files will be restricted because they may contain information that would be embarrassing or damaging to members or staff. These will require special permission from the Board of Trustees for access.

The future:

Starting in December 2015 the Historians Task Force will be scanning and processing written documents into electronic format. These files will be stored on computer hard disk temporarily. We will be doing the oldest records first. This is estimated to take at least 6 months.  After that the files will be updated monthly.

Phase II of the project will be processing photographs for the system.

Phase III of the project will be processing audio and video files. (I hope)

Wish us luck. We may need it.

 

Gary Roach

UUCLB Historian
